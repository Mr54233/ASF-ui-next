## 开发建议

1. **从 P0 功能开始** - WebSocket 和 ConfigEditor 是最核心的基础设施
2. **参考老版本实现** - 所有功能逻辑都可以从老版本复制
3. **TypeScript 重写** - 在老版本逻辑基础上添加类型定义
4. **组件化改造** - 将 Options API 改为 Composition API
5. **测试覆盖** - 每个功能完成后添加测试
